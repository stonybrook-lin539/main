Tiers are a nice linguistic metaphor, but what is going on here at a formal level? How do we specify tiers, and how significant a modification are tiers? Also, what exactly is the cognitive intuition behind tiers? These are great questions with some great answers.

Tier projection as a mapping

Mathematically, tiers are easily defined as the output of a tier projection function \(\tau\) (this is the Greek letter tau). Each type of tier comes with its own projection function \(\tau_T\), which takes a string s as its only argument. Here \(T\) is a tier alphabet, i.e. a set of symbols that should be projected onto said tier. The tier projection function \(\tau_T\) then deletes every symbol from string s that isn’t a member of \(T\).

Given alphabets \(\Sigma\) and \(T\), the tier projection function \(\tau_T: \Sigma^* \rightarrow T^*\) maps each \(\Sigma\)-string \(s\) to the longest subsequence \(u\) of \(s\) such that \(u\) is a \(T\)-string.

Consider once more the case of Samala sibilant harmony. Since the set of sibilants contains only s and ʃ, the tier alphabet \(T\) for the sibilant tier contains those two symbols and nothing else. And \(\tau_T(\mathit{hasxintilawas}) = \mathit{ss}\).

Explain why the domain of \(\tau_T\) is \(\Sigma^*\) while its co-domain is \(T^*\).

Look at the list below, which shows \(\Sigma\)-strings being mapped to \(T\)-strings by some tier projection function. Write down the smallest possible tier alphabet \(T\) that gives rise to these mappings.

  • \(\mathit{aaca} \mapsto \mathit{aaa}\)
  • \(\mathit{cabab} \mapsto \mathit{abab}\)
  • \(\mathit{bbdbe} \mapsto \mathit{bbdbe}\)

Now that we have a formal way to specify tiers, it is very easy to graft that onto \(n\)-gram grammars by tying \(n\)-grams to specific tiers.

A tier \(n\)-gram grammar over alphabet \(\Sigma\) is a finite set of pairs \(\left \langle g_i, T_i \right \rangle\) (\(m \geq 0\)) such that

  • each \(T_i\) is a finite set of symbols drawn from \(\Sigma\),
  • \(g_i \in T_i^n\).

We call \(T\) a tier of \(G\) iff \(\left \langle g, T \right \rangle \in G\) for some \(g\). A \(\Sigma\)-string \(s\) is well-formed with respect to

  • the negative interpretation of \(G\) iff there is no \(\left \langle g,T \right \rangle \in G\) such that \(g\) is a factor of \(\tau_T(s)\),
  • the positive interpretation of \(G\) iff for every tier \(T\) of \(G\) and factor \(g\) of \(\tau_T(s)\), it holds that \(\left \langle g, T \right \rangle \in G\).

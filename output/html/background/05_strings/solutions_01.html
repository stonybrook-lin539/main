Substrings

For each one of the gaps below, enter \(\sqsubseteq\), \(\sqsubsetneq\), or \(\not\sqsubseteq\) depending on whether the first string is a substring of the second string, a proper substring, or neither:

  1. \(\mathit{a} \_ \mathit{aaaa}\)
  2. \(\mathit{a} \_ \mathit{b}\)
  3. \(\varepsilon\_ \mathit{b}\)
  4. \(\varepsilon\_ \varepsilon\)
  5. \(\mathit{aa} \_ \mathit{abbbca}\)
  6. \(\mathit{bc} \_ \mathit{abbbca}\)
  7. \(\mathit{cb} \_ \mathit{abbbca}\)
  1. \(\mathit{a} \sqsubsetneq \mathit{aaaa}\)
  2. \(\mathit{a} \not\sqsubseteq \mathit{b}\)
  3. \(\varepsilon\sqsubsetneq \mathit{b}\)
  4. \(\varepsilon\sqsubseteq \varepsilon\)
  5. \(\mathit{aa} \not\sqsubseteq \mathit{abbbca}\)
  6. \(\mathit{bc} \sqsubsetneq \mathit{abbbca}\)
  7. \(\mathit{cb} \not\sqsubseteq \mathit{abbbca}\)

Subsequence

For each one of the gaps below, enter \(\sqsubseteq\), \(\sqsubsetneq\), or \(\not\sqsubseteq\) depending on whether the first string is a subsequence of the second string, a proper subsequence, or neither:

  1. \(\mathit{a} \_ \mathit{aaaa}\)
  2. \(\mathit{a} \_ \mathit{b}\)
  3. \(\varepsilon\_ \mathit{b}\)
  4. \(\varepsilon\_ \varepsilon\)
  5. \(\mathit{aa} \_ \mathit{abbbca}\)
  6. \(\mathit{bc} \_ \mathit{abbbca}\)
  7. \(\mathit{cb} \_ \mathit{abbbca}\)
  1. \(\mathit{a} \sqsubsetneq \mathit{aaaa}\)
  2. \(\mathit{a} \not\sqsubseteq \mathit{b}\)
  3. \(\varepsilon\sqsubsetneq \mathit{b}\)
  4. \(\varepsilon\sqsubseteq \varepsilon\)
  5. \(\mathit{aa} \sqsubsetneq \mathit{abbbca}\)
  6. \(\mathit{bc} \sqsubsetneq \mathit{abbbca}\)
  7. \(\mathit{cb} \not\sqsubseteq \mathit{abbbca}\)

Say whether the following is True or False: Every substring of some string \(s\) is also a subsequence of \(s\), but not the other way round. Justify your answer.

This is correct. Suppose \(u \mathrel{\mathop:}=u_1 \cdots u_n\) is a substring of some string \(s\). Then it must be the case that \(u_1\) appears before \(u_2, u_3, \ldots, u_n\), \(u_2\) appears before \(u_3, \ldots, u_n\), and so on. But this is all that is required for \(u_1 \cdots u_n\) to be a subsequence of \(s\).

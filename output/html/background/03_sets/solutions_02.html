Union, intersection, and complement of sets (Solutions)

Compute the union of the following:

  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cup \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cup \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,1 \right \}\)
  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} = \left \{ 0,1,2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \left \{ 1,2,3 \right \} = \left \{ 0,1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cup \left \{ 0,1 \right \} = \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cup \emptyset = \left \{ 0,1 \right \}\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,1 \right \} = \left \{ 0,1,2,3 \right \}\)
  1. The union must contain all elements of the first set, all elements of the second set, and nothing else. In this case, where the first set contains \(0\) and \(1\), and the second contains \(2\) and \(3\), the union hence must contain all of the following, and nothing else:\(0\), \(1\), \(2\), and \(3\).
  2. The same logic still applies. The fact that \(1\) appears in both sets does not affect how union works.
  3. The union of two identical sets always returns the same set.
  4. Since the empty set contains no elements at all, it is always the case \(A \cup \emptyset = A\) (and \(\emptyset \cup A = A\)), no matter what \(A\) looks like. And yes, this means that \(\emptyset = \emptyset \cup \emptyset\).
  5. This is exactly the same as the second exercise because union does not care about the order of the sets: \(A \cup B\) is always the same as \(B \cup A\), just like \(m + n\) is always the same as \(n + m\) for addition of numbers.

Compute the union of the following in a step-wise fashion:

  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} \cup \emptyset\)
  2. \(\left \{ 0,1 \right \} \cup \emptyset \cup \left \{ 2,3 \right \}\)
  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} \cup \emptyset = (\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}) \cup \emptyset = \left \{ 0,1,2,3 \right \} \cup \emptyset = \left \{ 0,1,2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \emptyset \cup \left \{ 2,3 \right \} = (\left \{ 0,1 \right \} \cup \emptyset) \cup \left \{ 2,3 \right \} = \left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} = \left \{ 0,1,2,3 \right \}\)

Another way to phrase the exercise would be to ask you to verify that \((\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}) \cup \emptyset = \left \{ 0,1 \right \} \cup (\left \{ 2,3 \right \} \cup \emptyset)\). Either way you are verifying that it does not matter in what order you carry out the union steps, the result will always be the same.

Compute the intersection of the following:

  1. \(\left \{ 0,1 \right \} \cap \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cap \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cap \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \}\)
  1. \(\left \{ 0,1 \right \} \cap \left \{ 2,3 \right \} = \emptyset\)
  2. \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \} = \left \{ 1 \right \}\)
  3. \(\left \{ 0,1 \right \} \cap \left \{ 0,1 \right \} = \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cap \emptyset = \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \} = \left \{ 1 \right \}\)
  1. As neither set contains any elements that occur in the other set, their intersection is empty.
  2. The only element shared by both sets is \(1\), hence the intersection is \(\left \{ 1 \right \}\). Note that the intersection of two sets is always a set, even if that set has only one member. It is incorrect to say that the intersection of \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\) is \(1\), it is the set \(\left \{ 1 \right \}\).
  3. Just like \(A \cup A = A\) is universally true no matter what this set \(A\) looks like, it is always the case that \(A \cap A = A\).
  4. Since the empty set contains no elements at all, the result of intersecting a set \(A\) with the empty set will always be the empty set, no matter what \(A\) looks like.
  5. When we contrast that against the second exercise with \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\), we see once again that the order of the sets does not matter for intersection: \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \} = \left \{ 1 \right \} = \left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \}\). Just like union, intersection is commutative.

Give a concrete example where \(A - B = B - A\). Then make a single change to \(A\) such that \(A - B \neq B - A\).

Let \(A = B = \left \{ 1 \right \}\). Then \(A - B = \left \{ 1 \right \} - \left \{ 1 \right \} = B - A\). Now remove \(1\) from \(B\) such that \(B = \emptyset\). Then \(A - B = \left \{ 1 \right \} - \emptyset = \left \{ 1 \right \}\), which is distinct from \(B - A = \emptyset - \left \{ 1 \right \} = \emptyset\).

In general, \(A - B \neq B - A\) whenever \(A\) and \(B\) are distinct sets, but if \(A = B\) then \(A - B\) is always the same as \(B - A\). Hence we can always pick \(A\) and \(B\) to be identical sets and then make a single change to render them distinct.

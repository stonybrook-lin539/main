Union, intersection, and complement of sets (Solutions)

For each one of the statements below, say whether it is true or false.

  • \(\emptyset = \left \{ \right \}\)
  • \(\emptyset = \left \{ \emptyset \right \}\)
  • \(\emptyset \in \left \{ a \right \}\)
  • \(\left \{ a,b \right \} \neq \emptyset\)
  • True
  • False
  • False
  • True
  • By definition, \(\emptyset\) and \(\left \{ \right \}\) are just different ways of denoting the empty set.
  • The empty set is not the same thing as the set containing the empty set. The former has no elements, whereas the latter has exactly one element (which happens to be the empty set, but this is irrelevant).
  • The set \(\left \{ a \right \}\) has exactly one member, which is \(a\). When using symbols like \(a\) or \(b\) in exercises, we usually assume that they are distinct from other symbols. So we may assume that \(a \neq \emptyset\), which means that \(\emptyset \notin \left \{ a \right \}\).
  • The set \(\left \{ a,b \right \}\) has two members (again we assume that \(a \neq b\)), whereas the empty set has no members at all. Clearly, then, these must be distinct sets.

Compute the union of the following:

  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cup \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cup \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,1 \right \}\)
  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} = \left \{ 0,1,2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \left \{ 1,2,3 \right \} = \left \{ 0,1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cup \left \{ 0,1 \right \} = \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cup \emptyset = \left \{ 0,1 \right \}\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,1 \right \} = \left \{ 0,1,2,3 \right \}\)
  1. The union must contain all elements of the first set, all elements of the second set, and nothing else. In this case, where the first set contains \(0\) and \(1\), and the second contains \(2\) and \(3\), the union hence must contain all of the following, and nothing else: \(0\), \(1\), \(2\), and \(3\).
  2. The same logic still applies. The fact that \(1\) appears in both sets does not affect how union works.
  3. The union of two identical sets always returns the same set.
  4. Since the empty set contains no elements at all, it is always the case that \(A \cup \emptyset = A\) (and \(\emptyset \cup A = A\)), no matter what \(A\) looks like. And yes, this means that \(\emptyset = \emptyset \cup \emptyset\).
  5. This is exactly the same as the second exercise because union does not care about the order of the sets: \(A \cup B\) is always the same as \(B \cup A\), just like \(m + n\) is always the same as \(n + m\) for addition of numbers.

Compute the union of the following:

  1. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,\left \{ 1 \right \} \right \}\)
  1. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 2,3 \right \} = \left \{ 0, \left \{ 1 \right \}, 2, 3 \right \}\)
  2. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 1,2,3 \right \} = \left \{ 0, \left \{ 1 \right \}, 1, 2, 3 \right \}\)
  3. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 0,1 \right \} = \left \{ 0, \left \{ 1 \right \}, 1 \right \}\)
  4. \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \emptyset = \left \{ 0, \left \{ 1 \right \} \right \}\)
  5. \(\left \{ 1,2,3 \right \} \cup \left \{ 0,\left \{ 1 \right \} \right \} = \left \{ 1,2,3,0,\left \{ 1 \right \} \right \}\)
  1. Again we keep all elements from both sets. The only wrinkle is that one of those elements is itself a set, but that’s irrelevant for union. We do not care what exactly the elements are, if it is an element of at least one of \(A\) and \(B\) then it must be in \(A \cup B\).
  2. The important thing to keep in mind here is that \(1 \neq \left \{ 1 \right \}\), so the union must contain both \(1\) and \(\left \{ 1 \right \}\).
  3. For the same reason as above, the union must contain both \(1\) and \(\left \{ 1 \right \}\). If the exercise had asked for the union of \(\left \{ 0, \left \{ 1 \right \} \right \} \cup \left \{ 0, \left \{ 1 \right \} \right \}\), then the union would have been \(\left \{ 0, \left \{ 1 \right \} \right \}\) because \(A \cup A = A\) for every set \(A\). Note again that it is completely immaterial for all of this that \(\left \{ 1 \right \}\) is a set, the only thing that matters is that it is an element of at least one set and is distinct from \(0\) and \(1\).
  4. As always, \(A \cup \emptyset = A\) no matter what \(A\) looks like.
  5. This is again the same \(\left \{ 0,\left \{ 1 \right \} \right \} \cup \left \{ 1,2,3 \right \} = \left \{ 0, \left \{ 1 \right \}, 1, 2, 3 \right \}\). Note that the solution given here has a different order from that set, but since sets have no internal order this is just an artefact of our writing system.

Compute the union of the following in a step-wise fashion:

  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} \cup \emptyset\)
  2. \(\left \{ 0,1 \right \} \cup \emptyset \cup \left \{ 2,3 \right \}\)
  1. \(\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} \cup \emptyset = (\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}) \cup \emptyset = \left \{ 0,1,2,3 \right \} \cup \emptyset = \left \{ 0,1,2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cup \emptyset \cup \left \{ 2,3 \right \} = (\left \{ 0,1 \right \} \cup \emptyset) \cup \left \{ 2,3 \right \} = \left \{ 0,1 \right \} \cup \left \{ 2,3 \right \} = \left \{ 0,1,2,3 \right \}\)

Another way to phrase the exercise would be to ask you to verify that \((\left \{ 0,1 \right \} \cup \left \{ 2,3 \right \}) \cup \emptyset = \left \{ 0,1 \right \} \cup (\left \{ 2,3 \right \} \cup \emptyset)\). Either way you are verifying that it does not matter in what order you carry out the union steps as the result will always be the same.

Compute the intersection of the following:

  1. \(\left \{ 0,1 \right \} \cap \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,1 \right \} \cap \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cap \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \}\)
  1. \(\left \{ 0,1 \right \} \cap \left \{ 2,3 \right \} = \emptyset\)
  2. \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \} = \left \{ 1 \right \}\)
  3. \(\left \{ 0,1 \right \} \cap \left \{ 0,1 \right \} = \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,1 \right \} \cap \emptyset = \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \} = \left \{ 1 \right \}\)
  1. As neither set contains any elements that occur in the other set, their intersection is empty.
  2. The only element shared by both sets is \(1\), hence the intersection is \(\left \{ 1 \right \}\). Note that the intersection of two sets is always a set, even if that set has only one member. It is incorrect to say that the intersection of \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\) is \(1\), it is the set \(\left \{ 1 \right \}\).
  3. Just like \(A \cup A = A\) is universally true no matter what this set \(A\) looks like, it is always the case that \(A \cap A = A\).
  4. Since the empty set contains no elements at all, the result of intersecting a set \(A\) with the empty set will always be the empty set, no matter what \(A\) looks like. Commit that to memory: \(A \cap \emptyset = \emptyset\) for every set \(A\).
  5. When we contrast that against the second exercise with \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \}\), we see once again that the order of the sets does not matter for intersection: \(\left \{ 0,1 \right \} \cap \left \{ 1,2,3 \right \} = \left \{ 1 \right \} = \left \{ 1,2,3 \right \} \cap \left \{ 0,1 \right \}\). Just like union, intersection is commutative.

Compute the intersection of the following:

  1. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 2,3 \right \}\)
  2. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 1,2,3 \right \}\)
  3. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 0,1 \right \}\)
  4. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,\left \{ 1 \right \} \right \}\)
  1. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 2,3 \right \} = \emptyset\)
  2. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 1,2,3 \right \} = \emptyset\)
  3. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \left \{ 0,1 \right \} = \left \{ 0 \right \}\)
  4. \(\left \{ 0,\left \{ 1 \right \} \right \} \cap \emptyset = \emptyset\)
  5. \(\left \{ 1,2,3 \right \} \cap \left \{ 0,\left \{ 1 \right \} \right \} = \emptyset\)
  1. As the two sets have no elements in common, their intersection is empty.
  2. Again it is crucial to keep in mind that \(\left \{ 1 \right \} \neq 1\). Hence the two sets have no overlap and their intersection is the empty set.
  3. Since \(\left \{ 1 \right \} \neq 1\), only \(0\) is a member of both sets, and the intersection is \(\left \{ 0 \right \}\). Remember that the intersection is not just \(0\), but \(\left \{ 0 \right \}\). Intersection of two sets always yields a set.
  4. This must be the empty set because \(A \cap \emptyset = \emptyset\) for every set \(A\).
  5. Since \(A \cap B = B \cap A\), we can just reuse the result from the second line of this exercise.

Give a concrete example where \(A - B = B - A\). Then make a single change to \(A\) such that \(A - B \neq B - A\).

Let \(A = B = \left \{ 1 \right \}\). Then \(A - B = \left \{ 1 \right \} - \left \{ 1 \right \} = B - A\). Now remove \(1\) from \(B\) such that \(B = \emptyset\). Then \(A - B = \left \{ 1 \right \} - \emptyset = \left \{ 1 \right \}\), which is distinct from \(B - A = \emptyset - \left \{ 1 \right \} = \emptyset\).

In general, \(A - B \neq B - A\) whenever \(A\) and \(B\) are distinct sets, but if \(A = B\) then \(A - B\) is always the same as \(B - A\). Hence we can always pick \(A\) and \(B\) to be identical sets and then make a single change to render them distinct.

Assume that the context fixes \(A \mathrel{\mathop:}=\left \{ 0,1,2,3 \right \}\). Compute all of the following:

  1. \(\overline{\left \{ 0 \right \}}\)
  2. \(\overline{\left \{ 0,\left \{ 1 \right \} \right \}}\)
  3. \(\overline{\left \{ \right \}}\)
  4. \(\overline{\left \{ \left \{ 0 \right \} \right \}}\)
  5. \(\left \{ \overline{\left \{ 0 \right \}} \right \}\)
  1. \(\overline{\left \{ 0 \right \}} = \left \{ 1,2,3 \right \}\)
  2. \(\overline{\left \{ 0,\left \{ 1 \right \} \right \}} = \left \{ 1,2,3 \right \}\)
  3. \(\overline{\left \{ \right \}} = \left \{ 0,1,2,3 \right \}\)
  4. \(\overline{\left \{ \left \{ 0 \right \} \right \}} = \left \{ 0,1,2,3 \right \}\)
  5. \(\left \{ \overline{\left \{ 0 \right \}} \right \} = \left \{ \left \{ 1,2,3 \right \} \right \}\)
  1. Given the context specified at the beginning of the exercise, we may rewrite \(\overline{\left \{ 0 \right \}}\) as \(\left \{ 0,1,2,3 \right \} - \left \{ 0 \right \}\). This is the set of all elements in \(\left \{ 0,1,2,3 \right \}\) that are not elements of \(\left \{ 0 \right \}\). The only such elements are \(1\), \(2\), and \(3\), and thus the set is \(\left \{ 1,2,3 \right \}\).
  2. Keep in mind that \(1 \neq \left \{ 1 \right \}\). So the only element of \(\left \{ 0,1,2,3 \right \}\) that isn’t a member of \(\left \{ 0,\left \{ 1 \right \} \right \}\) is \(0\), and we once again get \(\left \{ 1,2,3 \right \}\).
  3. Since the empty set contains no elements at all, we do not remove any elements at all from \(\left \{ 0,1,2,3 \right \}\). Remember: \(A - \emptyset = A\) for every set \(A\).
  4. This formula is just a different way of writing \(\left \{ 0,1,2,3 \right \} - \left \{ \left \{ 0 \right \} \right \}\). But no member of \(\left \{ 0,1,2,3 \right \}\) is a member of \(\left \{ \left \{ 0 \right \} \right \}\) (remember that \(0 \neq \left \{ 0 \right \}\)). Hence we get \(\left \{ 0,1,2,3 \right \}\).
  5. This is a tricky one. You have to look very carefully to notice that this isn’t \(\overline{\left \{ \left \{ 0 \right \} \right \}}\) but rather \(\left \{ \overline{\left \{ 0 \right \}} \right \}\), which is the set that contains \(\overline{\left \{ 0 \right \}}\). We already know that the latter is \(\left \{ 1,2,3 \right \}\), so \(\left \{ \overline{\left \{ 0 \right \}} \right \}\) must be \(\left \{ \left \{ 1,2,3 \right \} \right \}\).

No matter how one chooses \(A\) and \(B\), it always holds that the complement of the complement of \(B\) is identical to \(B\) itself. In symbols: \(B = \overline{\overline{B}}\). Explain why this holds.

Hint: By definition, \(\overline{\overline{B}} = A - \overline{B} = A - (A - B)\).

At first glance, this seems to be wrong. For example, if \(A \mathrel{\mathop:}=\left \{ 1 \right \}\) and \(B \mathrel{\mathop:}=\left \{ 2 \right \}\), then \(\overline{B} = A - B = \left \{ 1 \right \} - \left \{ 2 \right \} = \left \{ 1 \right \}\), whereas \(\overline{\overline{B}} = A - (A - B) = \left \{ 1 \right \} - \left \{ 1 \right \} = \emptyset\). But there is an additional assumption in place when writing \(\overline{B}\) instead of \(A - B\). We said that \(\overline{B}\) is a shorthand for \(A - B\) if \(A\) is clear from context. But in addition, it must also be the case that \(B\) can only contain elements that are also elements of \(A\). In a sense, \(A\) serves as a fixed universe that specifies all possible elements, and other sets only get to choose from that universe.

With this additional assumption in place, the claim in the exercise is correct. Let us use \(S\) to denote \(A - B\). That is to say, \(S\) is the set of all \(x\) such that \(x \in A\) and \(x \notin B\). Hence \(A - S\) must be the set of all \(y\) such that \(y \in A\) and it is not the case that \(y \notin B\). This is the same thing as saying that \(S\) contains all \(y\) such that \(y \in A\) and \(y \in B\). But since we just said that the notation \(\overline{B}\) implies that every member of \(B\) is a member of \(A\), the \(y \in A\) part is redundant and we can shorten the definition of \(S\) to “the set of all \(y\) such that \(y \in B\)”. And that is simply \(B\).

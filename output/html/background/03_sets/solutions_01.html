Sets: Additional notation (Solutions)

For each one of the statements below, say whether it is true or false.

  1. \(\emptyset = \left \{ \right \}\)
  2. \(\emptyset \in \left \{ a \right \}\)
  3. \(\left \{ a,b \right \} \neq \emptyset\)
  1. \(\emptyset = \left \{ \right \}\) is true
  2. \(\emptyset \in \left \{ a \right \}\) is false
  3. \(\left \{ a,b \right \} \neq \emptyset\) is true
  1. This is correct, \(\emptyset\) is just a different way to refer to \(\left \{ \right \}\), the set without any members.
  2. The set \(\left \{ a \right \}\) only contains the element \(a\), it does not contain \(\emptyset\). If it did, the set would be written \(\left \{ \emptyset, a \right \}\). Hence \(\emptyset \notin \left \{ a \right \}\).
  3. Two sets \(A\) and \(B\) are distinct if there is at least one element \(x\) such that \(x \in A\) but \(x \notin B\) (which we can also write as \(A \ni x \notin B\)). In our case, there are two such elements, as \(\left \{ a,b \right \} \ni a \notin \emptyset\) and \(\left \{ a,b \right \} \ni b \notin \emptyset\).

For each one of the following, put \(=\) or \(\neq\) in the gap as appropriate:

  1. \(\left \{ 0,1,2,3,4,5 \right \} \_ \left \{ 10 - x \mid x \in \left \{ 5,6,7,8,9,10 \right \} \right \}\)
  2. \(\emptyset \_ \left \{ x \mid x \text{ is both a square and a circle} \right \}\)
  3. \(\left \{ 2,3,5,7 \right \} \_ \left \{ n + 1 \mid n \text{ is a prime number between 0 and 10} \right \}\)
  1. \(\left \{ 0,1,2,3,4,5 \right \} = \left \{ 10 - x \mid x \in \left \{ 5,6,7,8,9,10 \right \} \right \}\)
  2. \(\emptyset = \left \{ x \mid x \text{ is both a square and a circle} \right \}\)
  3. \(\left \{ 2,3,5,7 \right \} \neq \left \{ n + 1 \mid n \text{ is a prime number between 0 and 10} \right \}\)

  1. The set \(\left \{ 10 - x \mid x \in \left \{ 5, 6, 7, 8, 9, 10 \right \} \right \}\) contains \(5\) (\(10 - 5\)), \(4\) (\(10 - 6\)), \(3\) (\(10 - 7\)), \(2\) (\(10 - 8\)), \(1\) (\(10 - 9\)), and \(0\) (\(10 - 10\)), That is indeed the same as \(\left \{ 0,1,2,3,4,5, \right \}\).

  2. There is no \(x\) both a square and a circle, hence the set \(\left \{ x \mid x \text{ is both a square and a circle} \right \}\) is empty and thus equivalent to \(\emptyset\).

  3. The prime numbers between 0 and 10 are 2, 3, 5, and 7 (remember, 1 is not a prime number). The set \(\left \{ n + 1 \mid n \text{ is a prime number between 0 and 10} \right \}\) thus consists of \(3\) (\(2 + 1\)) \(4\) (\(3 + 1\)) \(6\) (\(5 + 1\)), and \(8\) (\(7 + 1\)). But \(\left \{ 3,4,6,8 \right \} \neq \left \{ 2,3,5,7 \right \}\).

Express the following in terms of set-builder notation:

  1. \(\left \{ \text{George Washington}, \text{John Adams}, \text{Thomas Jefferson} \right \}\)
  2. \(\left \{ \text{a}, \text{e}, \text{i}, \text{o}, \text{u} \right \}\)
  1. \(\left \{ x \mid x \text{ is one of the first three presidents of the United States} \right \}\)
  2. \(\left \{ x \mid x \text{ is a vowel of the English alphabet} \right \}\)

  1. Depending on how well you know your US history, this one might have required a bit of research. But once you know what these three names have in common and what separates them from everything else, you can easily talk about the set of all \(x\) such that \(x\) is one of the first three presidents of the United States. The rest is just a matter of transforming that English statement into set-builder notation.

    Note that there are many other ways to pick out the same set. For instance, the set \(\left \{ x \mid x \text{ was a US president before James Madison} \right \}\) would describe the same set because Jame Madison was the fourth president. So one and the same set can be defined in many different ways via set-builder notation.

    In addition, the exercise doesn’t actually tell us what exactly the set is supposed to contain. Is it the president’s themselves, or strings that correspond to these presidents’ names? Depending on your interpretation, this also gives us slightly different definitions.

  2. Again there are many different possible definitions (and the one given is a little iffy anyways as English also uses y for vowels). Instead, we could have defined this as the set of characters that correspond to the numbers 97, 101, 105, 111, and 117 of the ASCII text encoding. Or the set of letters that are said at specific timestamps of the ABC song. Or the set of letters that you like best (assuming that those are actually a, e, i, o, and u). There’s infinitely many solutions, the important thing is whether yours uses the set-builder notation correctly.

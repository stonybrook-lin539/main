The Powerset (Solutions)

For each one of the following sets, compute its powerset.

  1. \(\left \{ a,b \right \}\)
  2. \(\left \{ a,b,c,d \right \}\)
  3. \(\left \{ \left \{ a \right \} \right \}\)
  4. \(\emptyset\)
  5. \(\left \{ \emptyset \right \}\)
  6. \(\wp(\left \{ \left \{ a \right \} \right \})\)
  7. \(\wp(\wp(\emptyset))\)
  1. \(\wp(\left \{ a,b \right \}) = \left \{ \emptyset, \left \{ a \right \}, \left \{ b \right \}, \left \{ a,b \right \} \right \}\)
  2. \(\wp(\left \{ a,b,c,d \right \})\) contains the following 16 elements
    1. \(\left \{ a,b,c,d \right \}\)
    2. \(\left \{ a,b,c \right \}\)
    3. \(\left \{ a,b,d \right \}\)
    4. \(\left \{ a,c,d \right \}\)
    5. \(\left \{ b,c,d \right \}\)
    6. \(\left \{ a,b \right \}\)
    7. \(\left \{ a,c \right \}\)
    8. \(\left \{ a,d \right \}\)
    9. \(\left \{ b,c \right \}\)
    10. \(\left \{ b,d \right \}\)
    11. \(\left \{ c,d \right \}\)
    12. \(\left \{ a \right \}\)
    13. \(\left \{ b \right \}\)
    14. \(\left \{ c \right \}\)
    15. \(\left \{ d \right \}\)
    16. \(\emptyset\)
  3. \(\wp(\left \{ \left \{ a \right \} \right \}) = \left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \}\)
  4. \(\wp(\emptyset) = \left \{ \emptyset \right \}\)
  5. \(\wp(\left \{ \emptyset \right \}) = \left \{ \emptyset, \left \{ \emptyset \right \} \right \}\)
  6. \(\wp(\wp(\left \{ \left \{ a \right \} \right \})) = \wp( \left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \}) = \left \{ \emptyset, \left \{ \emptyset \right \}, \left \{ \left \{ \left \{ a \right \} \right \} \right \}, \left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \} \right \}\)
  7. \(\wp(\wp(\wp(\emptyset))) = \wp(\wp( \left \{ \emptyset \right \})) = \wp ( \left \{ \emptyset, \left \{ \emptyset \right \} \right \} )\), which is the set containing all of the following (and nothing else):
    1. \(\emptyset\)
    2. \(\left \{ \emptyset \right \}\),
    3. \(\left \{ \left \{ \emptyset \right \} \right \}\),
    4. \(\left \{ \emptyset, \left \{ \emptyset \right \} \right \}\).
  1. The subsets of \(\left \{ a,b \right \}\) are the empty set (as it is a subset of every set), \(\left \{ a \right \}\), \(\left \{ b \right \}\), and \(\left \{ a,b \right \}\).
  2. Again all we have to do is list all subsets and construct a set that contains all those sets, and nothing else.
  3. This one might be a little confusing because we now have a set inside the set whose powerset we want to construct. However, remember that the powerset of some set \(A\) is just the set of all subsets of \(A\), and for that it does not matter what exactly the elements of \(A\) look like. If you still find it confusing to have sets as elements, you can use the following trick: suppose \(s \mathrel{\mathop:}=\left \{ a \right \}\) so that we can write \(\left \{ \left \{ a \right \} \right \}\) simply as \(\left \{ s \right \}\). Then \(\wp(\left \{ s \right \})\) is \(\left \{ \emptyset, \left \{ s \right \} \right \}\). Now we go back and replace \(s\) with \(\left \{ a \right \}\) following our initial definition, yielding \(\left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \}\).
  4. This is where things can get confusing if you don’t pay attention. The powerset of \(\emptyset\) is the set of all subsets of \(\emptyset\). But the only subset of \(\emptyset\) is \(\emptyset\) itself, so that \(\wp(\emptyset) = \left \{ \emptyset \right \}\).
  5. Now things get really confusing: \(\wp(\left \{ \emptyset \right \})\), how could anybody figure that out? Well, this case works exactly the same \(\wp(\left \{ \left \{ a \right \} \right \})\), except that we now have \(\emptyset\) instead of \(\left \{ a \right \}\). Again, what we care about is the elements of the set whose powerset we are constructing, now that those elements actually look like. So we could again do our substitution trick, with \(s \mathrel{\mathop:}=\emptyset\), and then we have \(\wp(\left \{ s \right \}) = \left \{ \emptyset, \left \{ s \right \} \right \}\), and substituting for \(s\), we get \(\left \{ \emptyset, \left \{ \emptyset \right \} \right \}\).
  6. God have mercy on us, \(\wp(\wp(\left \{ \left \{ a \right \} \right \}))\), nobody could possibly figure that out! Actually it’s pretty simple. We already know that \(\wp(\left \{ \left \{ a \right \} \right \})\) is \(\left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \}\). For the sake of clarity, let us do our substitution trick again: \(s \mathrel{\mathop:}=\emptyset\), and \(t \mathrel{\mathop:}=\left \{ \left \{ a \right \} \right \}\). Then \(\left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \} = \left \{ s,t \right \}\), and \(\wp(\left \{ s,t \right \}) = \left \{ \emptyset, \left \{ s \right \}, \left \{ t \right \}, \left \{ s,t \right \} \right \}\). Substitute for \(s\) and \(t\), and we get \(\left \{ \emptyset, \left \{ \emptyset \right \}, \left \{ \left \{ \left \{ a \right \} \right \} \right \}, \left \{ \emptyset, \left \{ \left \{ a \right \} \right \} \right \} \right \}\).
  7. Alright, but now the universe is surely about to implode, \(\wp(\wp(\wp(\emptyset)))\), that’s plain lunacy! Nope, once again straight-forward, we just have to take our time. We already know that \(\wp(\wp(\emptyset))\) is \(\left \{ \emptyset, \left \{ \emptyset \right \} \right \}\). We do our substitution trick one more time, now with \(s \mathrel{\mathop:}=\emptyset\) and \(t \mathrel{\mathop:}=\left \{ \emptyset \right \}\). Again we calculate \(\wp(\left \{ s,t \right \})\), and substituting for \(s\) and \(t\) we get \(\{ \emptyset, \left \{ \emptyset \right \}, \left \{ \left \{ \emptyset \right \} \right \}, \left \{ \emptyset, \left \{ \emptyset \right \} \right \}\}\).

Powerset notation

For each set \(A\) in the previous exercise, verify that \(\left | \wp(A) \right | = 2^{\left | A \right |}\).

  1. \(\left | \left \{ a,b \right \} \right | = 2\) and \(\left | \wp(\left \{ a,b \right \}) \right | = 2^2 = 4\)
  2. \(\left | \left \{ a,b,c,d \right \} \right | = 4\) and \(\left | \wp(\left \{ a,b,c,d \right \} \right | = 2^4 = 16\)
  3. \(\left | \left \{ \left \{ a \right \} \right \} \right | = 1\) and \(\left | \wp(\left \{ \left \{ a \right \} \right \} \right | = 2^1 = 2\)
  4. \(\left | \emptyset \right | = 0\) and \(\left | \wp(\emptyset) \right | = 2^0 = 1\)
  5. \(\left | \left \{ \emptyset \right \} \right | = 1\) and \(\left | \wp(\left \{ \emptyset \right \} \right | = 2^1 = 2\)
  6. \(\left | \wp(\left \{ \left \{ a \right \} \right \}) \right | = 2\) and \(\left | \wp(\wp(\left \{ \left \{ a \right \} \right \})) \right | = 2^2 = 4\)
  7. \(\left | \wp(\wp(\emptyset)) \right | = 2\) and \(\left | \wp(\wp(\wp(\emptyset))) \right | = 2^2 = 4\)

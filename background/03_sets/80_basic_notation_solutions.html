Sets: The basics (Solutions)

Write the following as a set:

  1. the first names of your three favorite actors/actresses,
  2. the colors of the rainbow,
  3. all even numbers between 1 and 11
  • \(\{\)Bruce, Marlene, Bela\(\}\)
  • \(\{\)red, orange, yellow, green, blue, indigo, violet\(\}\)
  • \(\left \{ 2,4,6,8,10 \right \}\)

  1. Suppose that your favorite actors are Bruce Campbell (Evil Dead), Marlene Dietrich (The Blue Angel, Witness for the Prosecution), and Bela Lugosi (Dracula). Their first names are Bruce, Marlene, and Bela, so the set that contains their first names, and nothing else, is \(\{\)Bruce, Marlene, Bela\(\}\).

  2. If you’re like me, you probably had to look this one up, but rainbows have six colors, which are red, orange, yellow, green, blue, indigo, and violet. So the set we’re looking for contains these six colors. We can specify that set using the names of these colors, but it would also be okay to use colored squares instead of names, colored lines, anything that conveys clearly that this is the set that contains these six colors.

  3. A number is even if and only if it is a multiple of 2. Between 1 and 11, that’s 2, 4, 6, 8, and 10.

Put \(\in\), \(\ni\), \(\notin\), \(\not\ni\) in the gaps below as appropriate:

  1. \(5 \_ \left \{ 1,2,4,5,8 \right \}\)
  2. \(6 \_ \left \{ 1,2,4,5,8 \right \}\)
  3. \(\left \{ 5 \right \} \_ \left \{ 1,2,4,5,8 \right \}\)
  4. \(5 \_ \left \{ 1,2,4,5,8 \right \} \_ 6\)
  1. \(5 \in \left \{ 1,2,4,5,8 \right \}\)
  2. \(6 \notin \left \{ 1,2,4,5,8 \right \}\)
  3. \(\left \{ 5 \right \} \notin \left \{ 1,2,4,5,8 \right \}\)
  4. \(5 \in \left \{ 1,2,4,5,8 \right \} \notin 6\) or \(5 \in \left \{ 1,2,4,5,8 \right \} \not\ni 6\).
  1. The set \(\left \{ 1,2,4,5,8 \right \}\) has 5 as one of its members, which we write as \(5 \in \left \{ 1,2,4,5,8 \right \}\).
  2. On the other hand, \(6\) is nowhere to be found in \(\left \{ 1,2,4,5,8 \right \}\), its only elements are \(1\), \(2\), \(4\), \(5\), and \(8\). Hence \(6 \notin \left \{ 1,2,4,5,8 \right \}\).
  3. This one is a bit tricky. We already know that \(\left \{ 1,2,4,5,8 \right \}\) contains \(5\), but that’s not the same thing as containing \(\left \{ 5 \right \}\), i.e. the set containing \(5\). None of the elements of \(\left \{ 1,2,4,5,8 \right \}\) is \(\left \{ 5 \right \}\). By the way, this is also why \(5 \notin \left \{ 1,2,4,\left \{ 5 \right \},8 \right \}\). Never confuse an object with a set containing that object, the two are very different things.
  4. We already know that \(\left \{ 1,2,4,5,8 \right \}\) contains \(5\), so the first gap must be \(\in\). For the second gap, there are two options. If we fill the gap with \(\notin\), we are saying that \(\left \{ 1,2,4,5,8 \right \}\) is not a member of \(6\), which is true, but it’s an odd thing to say as \(6\) isn’t a set to begin with. Instead, it makes more sense to fill the gap with \(\not\ni\), which states that \(6\) is not a member of \(\left \{ 1,2,4,5,8 \right \}\).

For each one of the following, fill the gap with \(=\) or \(\neq\) as appropriate:

  1. \(\left \{ a,b \right \} \_ \left \{ a,b \right \}\)
  2. \(\left \{ b,a \right \} \_ \left \{ a,b \right \}\)
  3. \(\left \{ b,a,c,d \right \} \_ \left \{ e,a,b,d \right \}\)
  1. \(\left \{ a,b \right \} = \left \{ a,b \right \}\)
  2. \(\left \{ b,a \right \} = \left \{ a,b \right \}\)
  3. \(\left \{ b,a,c,d \right \} \neq \left \{ e,a,b,d \right \}\) (assuming that \(c\) and \(e\) are distinct objects)
  1. Two sets are identical if they contain exactly the same elements. This is clearly the case here, both sets contain \(a\), both of them also contain \(b\), and they contain nothing else.
  2. It may seem like \(\left \{ b,a \right \}\) is different from \(\left \{ a,b \right \}\), but remember that sets are unordered. It does not matter in what order we write down the elements, all that matters is whether the sets have the same members. And this is still the case here.
  3. Each one of the sets contains four elements, three of which are \(a\), \(b\), and \(d\). Only the first set contains \(c\), and only the second set contains \(e\). This suggests that the sets are not equivalent. However, we have to be careful here as \(c\) and \(e\) may just be different ways of referring to the same object. As a concrete example, suppose that we are talking about sets of actors, and we are looking at the sets \(\{\)Arnold\(\}\) and \(\{\)Schwarzenegger\(\}\). These are not actually distinct sets, they are just different ways of writing down the set that contains only the actor Arnold Schwarzenegger. So remember: what matters isn’t how we choose to write down a set, what matters is what it actually contains. If both \(c\) and \(e\) are just ways of referring to, say, the number \(5\), then the two sets above are equivalent.

For each one of the following, fill the gap with \(=\) or \(\neq\) as appropriate:

  1. \(\left \{ a,b \right \} \_ \left \{ a,a,b,b \right \}\)
  2. \(\left \{ b,a \right \} \_ \left \{ a,b,a \right \}\)
  3. \(\left \{ c,b,a,a,d,c \right \} \_ \left \{ a,a,b,d,c,c,c \right \}\)
  4. \(\left \{ a \right \} \_ \left \{ a,a,a,a,a,a,c,a,a,a,a,a,a \right \}\)
  1. \(\left \{ a,b \right \} = \left \{ a,a,b,b \right \}\)
  2. \(\left \{ b,a \right \} = \left \{ a,b,a \right \}\)
  3. \(\left \{ c,b,a,a,d,c \right \} = \left \{ a,a,b,d,c,c,c \right \}\)
  4. \(\left \{ a \right \} \neq \left \{ a,a,a,a,a,a,c,a,a,a,a,a,a \right \}\)
  1. Remember, one and the same element cannot be contained in a set multiple times. Either it is a member of the set, or it is not a member, it cannot be a member two times, or three times, or anything like that. Hence \(\left \{ a,a,b,b \right \} = \left \{ a,b,b \right \} = \left \{ a,b \right \}\).
  2. The same logic applies in this case: \(\left \{ a,b,a \right \} = \left \{ a,b \right \}\), which is the same as \(\left \{ b,a \right \}\) because sets are not ordered.
  3. Again we have two identical sets. The first set contains \(a\), \(b\), \(c\), \(d\), and nothing else, and the same is true for the second set.
  4. Finally we have two sets that are distinct. The second set could be written more compactly as \(\left \{ a,c \right \}\), and that is not the same set as \(\left \{ a \right \}\) because only the former contains \(c\).

The sentence If police police police, then police police police actually uses two different word types. It just just so happens that both are pronounced and spelled police. But one is the noun police, the other one the verb police. So we might want to annotate the string as follows: If police[N] police[V] police[N], then police[N] police[V] police[N]. Assume that words are annotated with their part of speech in this fashion. Then what would be the corresponding set of words?

\(\left \{ \text{if}, \text{police[N]}, \text{police[V]}, \text{then} \right \}\).

We can build this set incrementally by moving from left to right through the string If police[N] police[V] police[N], then police[N] police[V] police[N] (ignoring capitalization). The string starts with if, so at this point our set is \(\left \{ \text{if} \right \}\). Next we see police[N], and since this is not in our set yet, we add it and obtain \(\left \{ \text{if}, \text{police[N]} \right \}\). After that we encounter police[V], which is not an element of our set yet — the set only contains the noun police, not the verb police. Hence we add police[V], yielding \(\left \{ \text{if}, \text{police[N]}, \text{police[V]} \right \}\). Now the next word is police[N], but since that’s already in our set we do not need to add it again (there’s no point in writing down the same element multiple times). After that we add then, and at this point we are done. Our set is \(\left \{ \text{if}, \text{police[N]}, \text{police[V]}, \text{then} \right \}\), and our example sentence contains no words after then that aren’t already members of this set.

For each one of the following, fill the gap with \(=\) or \(\neq\) as appropriate:

  1. \(\left \{ a,b \right \} \_ \left \{ a,a,b,\left \{ b \right \} \right \}\)
  2. \(\left \{ \left \{ b \right \},\left \{ a \right \} \right \} \_ \left \{ \left \{ a \right \},\left \{ b \right \},\left \{ a \right \} \right \}\)
  3. \(\left \{ c,b,\left \{ a,a \right \},d,c \right \} \_ \left \{ \left \{ a \right \},b,d,c,c,c \right \}\)
  4. \(\left \{ a \right \} \_ \left \{ a,a,a,a,a,a,\left \{ a \right \},a,a,a,a,a,a \right \}\)
  1. \(\left \{ a,b \right \} \neq \left \{ a,a,b,\left \{ b \right \} \right \}\)
  2. \(\left \{ \left \{ b \right \},\left \{ a \right \} \right \} = \left \{ \left \{ a \right \},\left \{ b \right \},\left \{ a \right \} \right \}\)
  3. \(\left \{ c,b,\left \{ a,a \right \},d,c \right \} = \left \{ \left \{ a \right \},b,d,c,c,c \right \}\)
  4. \(\left \{ a \right \} \neq \left \{ a,a,a,a,a,a,\left \{ a \right \},a,a,a,a,a,a \right \}\)
  1. Since \(b\) is not the same as \(\left \{ b \right \}\), \(\left \{ a,a,b,\left \{ b \right \} \right \}\) contains an element that is not a member of \(\left \{ a,b \right \}\), which means that the two are distinct sets.
  2. As sets do not contain duplicates, it holds that \(\left \{ \left \{ a \right \}, \left \{ b \right \}, \left \{ a \right \} \right \} = \left \{ \left \{ a \right \}, \left \{ b \right \} \right \}\). But sets are also unordered, so that \(\left \{ \left \{ a \right \}, \left \{ b \right \} \right \} = \left \{ \left \{ b \right \}, \left \{ a \right \} \right \}\).
  3. Again we have to keep in mind that sets do not contain duplicates, and as a result \(\left \{ a,a \right \} = \left \{ a \right \}\). We thus can simplify the two sets as follows: \(\left \{ c,b,\left \{ a,a \right \},d,c \right \} = \left \{ c,b,\left \{ a \right \},d,c \right \} = \left \{ c,b,\left \{ a \right \}, d \right \} = \left \{ \left \{ a \right \},b,c,d \right \}\) and \(\left \{ \left \{ a \right \},b,d,c,c,c \right \} = \left \{ \left \{ a \right \},b,d,c \right \} = \left \{ \left \{ a \right \},b,c,d \right \}\).
  4. We can reduce \(\left \{ a,a,a,a,a,a,\left \{ a \right \},a,a,a,a,a,a \right \}\) to \(\left \{ a, \left \{ a \right \} \right \}\), but since \(a \neq \left \{ a \right \}\), we can see that \(\left \{ a, \left \{ a \right \} \right \}\) is not the same set as \(\left \{ a \right \}\).

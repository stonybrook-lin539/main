# Cardinality of multisets

::: exercise
Calculate the cardinality of the following multisets:


- $\setof{a:17}$
- $\setof{\text{John}: 5, \text{Mary}: 5, \text{Bill}: 10}$
- $\setof{a: 0, b: 0, c:0}$

:::

::: solution

- $\card{\setof{a:17}} = 17$
- $\card{\setof{\text{John}: 5, \text{Mary}: 5, \text{Bill}: 10}} = 20$
- $\card{\setof{a: 0, b: 0, c:0}} = 0$

:::
